# Laravel Presets

Our default Laravel presets.


## Installation
Require the package using composer.
```bash
composer require code413/presets --dev
```

Publish any of the available presets.
```bash
php artisan ui blade 
```

## Available Presets
- blade
- bootstrap
- tabler
- tailwind

## Compatibility
| Laravel  | Code413/Presets |
|:---:|:---:|
| 5  | 1  |
| 6  | 1  |
| 7  | 2  |
